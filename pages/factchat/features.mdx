import TocSidebar, { extractHeadings } from '../../components/toc-sidebar'
import { useEffect, useState } from 'react'

export function TocWrapper() {
  const [headings, setHeadings] = useState([])

  useEffect(() => {
    // Extract headings after component mounts
    const tocHeadings = extractHeadings()
    setHeadings(tocHeadings)
  }, [])

  return <TocSidebar headings={headings} />
}

# FactChat Features Guide

FactChat offers a comprehensive suite of features designed to empower organizations with AI-driven chatbot capabilities. This guide covers all major features and how to use them effectively.

## Document Training

### Upload Documents

FactChat allows you to train your chatbot with up to 100 documents, ensuring domain-specific accuracy and relevance.

#### Supported File Formats

The platform supports various document formats including:
- PDF documents
- Word files (.doc, .docx)
- Text files (.txt)
- Markdown files (.md)
- CSV data files

#### Best Practices

For optimal training results, follow these recommendations:
- Ensure documents are well-structured with clear headings
- Remove sensitive information before uploading
- Use consistent terminology across documents
- Keep file sizes under 10MB for faster processing

### Training Process

The training process is automated and typically completes within minutes. The system analyzes your documents and creates a knowledge base that the chatbot uses to generate accurate responses.

## Integration Options

### Web Integration

Embed FactChat on your website with a simple code snippet.

#### JavaScript Embed

```javascript
<script src="https://factchat.example.com/embed.js"></script>
<script>
  FactChat.init({
    botId: 'your-bot-id',
    theme: 'light',
    position: 'bottom-right'
  })
</script>
```

#### WordPress Plugin

Install the FactChat WordPress plugin for seamless integration with your WordPress site.

### API Integration

FactChat provides a robust API for custom integrations.

#### Authentication

All API requests require authentication using API keys:

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
     https://api.factchat.example.com/v1/chat
```

#### Chat API

Send messages programmatically and receive AI-generated responses:

```json
{
  "message": "What are your business hours?",
  "conversationId": "optional-conversation-id",
  "context": {}
}
```
